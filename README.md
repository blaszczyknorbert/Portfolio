# norbert błaszczyk · portfolio

Personal portfolio site — [blaszczyknorbert.com](https://blaszczyknorbert.com)

## stack

Vanilla HTML/CSS/JS, no framework, no build step. Single `index.html` with self-hosted JetBrains Mono.

## structure

```
index.html        entry point
assets/og.png     open graph image
fonts/            self-hosted woff2 font files
```

## deployment

Hosted on Cloudflare Pages, served from the `main` branch.
